Что такое СП?

СП - это оригинальный проект в основе которого лежит ванильный сервер по Minecraft и полная свобода действий на нём. Только помните, что ваша свобода заканчивается там, где начинается свобода другого игрока. На наших серверах есть ПРАВИЛА и ЗАКОНЫ. Нарушая правила, а именно: Быть адекватным игроком, не гриферить, не мешать другим игрокам наслаждаться игрой, Вы будете забанены навсегда без суда. Нарушив законы, которые принимает правительство сервера, над вами будет проведён суд как в реальной жизни.

Если у тебя есть какие-то вопросы, ты можешь их задать в нашей группе ВКонтакте

Попробуй СП бесплатно

Тут можно пообщаться с друзьями, порыбачить, побывать на концерте и просто хорошо провести время. Для входа требуется Лицензионный Java Minecraft версии 1.16.5
